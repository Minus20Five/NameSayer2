What is NameSayer?
******************

The purpose of this program is for users to say unfamiliar names.
The user will be able to listen to the recordings stored in the name data base and record their own productions.
The user are allowed to practise full names that consist of partial names chosen from the name data base.

How to use NameSayer:
*********************
(1)Double click on the application ----> Main Menu
(2)Click on Browse Database to listen to the recordings stored in the name data base.
(3)Then choose the practise mode. There are two modes for practising:
			                                                      Practise Mode ----> practise as many times as you want but your recording cannot be saved to name data base
									      Assessment Mode ----> assess the user's pronunciation. (The user will be able to replay their recording then choose whether they want to save the recording to name data base or not.)
	
(4)Click on Practice Mode button ------> Name Selection Screen
The Name Selection Screen would contain:
										a search bar ----> search for a name
										a randomized toggle button -----> randomize the order of the selected list
										a next button -----> to enter the next stage of practsing
										a back button -----> in case the initialisation of names go off, back to step (1)
(5)Select one of the names displayed in the selected names list----> click play button to listen to recording in the database,click the record button to practise.
(6)In Assessment Mode the user are also allowed to:                       Click replay button ----> replay their recording and rate recording.
									  Click rateing button ---> rating stars are used to rate recordings.
									  Click save button ------> save the user's recording to name data base.


Use recordings
*********************
Once the user save their own recording, the production is saved to name database. The user can listen to this recording when browse name data base. And it will have a prefix of "Recording" indicating it is the users production.
